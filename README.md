# Portfolio Site

叶泽鸿的静态作品集站点。

当前站点包含：

- 首页 `index.html`
- 项目页 `digistudio.html`
- 项目页 `movie-visualization.html`
- 项目页 `digimi.html`
- 项目页 `readability-research.html`

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Local Preview

直接在浏览器打开 `index.html` 即可预览，或使用任意静态服务器在本地启动目录。

## Deploy

这个项目适合直接发布到 GitHub Pages。

推荐方式：

1. 将 `portfolio-site` 作为仓库根目录
2. 使用 `main` 分支
3. 在 GitHub Pages 中选择从仓库根目录发布
4. 保留根目录下的 `.nojekyll`
